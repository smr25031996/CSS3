Ways of CSS Styling

Inline

We can use the style attribute to specify the styling. This is referred to as inline styling. We can style only one element at a time using Inline styling

HTML and CSS

HTML stands for Hypertext markup language. HTML is a markup language for describing web pages.

CSS stands for cascading style sheets. CSS is used for styling the HTML Elements.

Internal

We can specify the style rules separately within the style tag in an HTML web page. This is referred to as internal/embedded styling as we embed the styling along with HTML Internal styling affects only the HTML elements of the webpage in which the internal style rules are written
